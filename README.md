# Clustering Analysis Using K-Means

## Project Overview
This project performs unsupervised learning using clustering techniques to identify patterns and group similar data points.


## Methodology
- Data preprocessing
- K-Means clustering
- Elbow Method to determine optimal clusters
- Silhouette score analysis
- Cluster visualization

## Results
Identified meaningful clusters and visualized group distributions.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
